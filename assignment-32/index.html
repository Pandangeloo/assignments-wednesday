Git Cheat Sheet



Kommando Beskrivning
git init Skapa lokalt nytt repository
git clone Klona exsisterande befintligt repository
git pull Ladda ner och lägga ihop "commits" från avlägsen repository
git add sätt upp alla förändringar
git commit -m "message" "Commit" alla uppsatta förändringar i repository
git push Putta lokala "commits" till avlägsna repository
git status Lista alla nya och modifierade files som kan "commitas"
git log Lista historiska varianter av den nuvarande branchen
git branch lista alla brancher i nuvarande repo
git branch < branch-name> Skapa en ny branch
